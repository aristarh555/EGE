# "Таблица Пифагора"

По данному натуральному числу n выведите таблицу Пифагора (таблицу умножения) размером n × n.

**Входные данные**

Натуральное число n.

**Выходные данные**

Таблица Пифагора. Числа в строке разделены пробелом.
