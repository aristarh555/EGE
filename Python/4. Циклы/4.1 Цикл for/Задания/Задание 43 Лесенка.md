# "Лесенка"

По данному натуральному числу n ≤ 9 выведите лесенку из n ступенек, i-я ступенька состоит из чисел от 1 до i без пробелов.

**Входные данные**

Натуральное число n.

**Выходные данные**

Лесенка.
